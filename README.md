# 个人技术网站

这是一个现代化的个人技术网站，展示了我的技术方向、项目经验和相关链接。

## 功能特性

- 响应式设计，适配各种设备
- 平滑滚动和动画效果
- 现代化的UI设计
- 包含技术方向展示
- 项目展示区域
- 外部链接（CSDN、GitHub、知乎）

## 文件结构

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 交互功能脚本

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome 图标库

## 设计亮点

- 使用CSS变量实现主题一致性
- Flexbox和Grid布局
- 平滑动画和过渡效果
- 移动端友好的导航菜单
- 可访问性优化（ARIA标签等）
