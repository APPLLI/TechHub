## 🚀 项目简介

TechHub是一个精心设计的个人技术展示网站，采用了现代化的前端技术栈，具有优雅的UI设计和流畅的用户体验。

## ✨ 功能特性

- 📱 **响应式设计** - 完美适配各种设备尺寸
- 🎨 **现代化UI** - 简洁优雅的界面设计
- ⚡ **平滑动画** - 流畅的滚动和过渡效果
- 📋 **技术方向** - 清晰展示专业技术领域
- 📁 **项目展示** - 突出重点项目和成果
- 🔗 **外部链接** - 集成CSDN、GitHub、知乎等平台
- 🌓 **主题切换** - 支持明暗两种主题模式

## 📁 文件结构


## 📦 安装使用

### 本地开发

1. 克隆仓库到本地
   ```bash
   git clone https://github.com/APPLLI/TechHub.git
   ```

2. 进入项目目录
   ```bash
   cd TechHub
   ```

3. 直接在浏览器中打开 `index.html` 文件

### 部署到GitHub Pages

项目已配置GitHub Pages，推送代码到 `main` 分支自动部署。
